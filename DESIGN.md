# Design System Document

 

## Theme: Fidelity

 

This document outlines the core visual characteristics of the "Fidelity" theme, providing a cohesive and accessible user experience.

 

### Color Palette
    
 

The color palette is built around a vibrant teal primary color, ensuring clarity and brand consistency.

 

*   **Primary Color:** `#39C5BB` - A bright, engaging teal used for primary actions and key interactive elements.

*   **Neutral Color:** `#F0F8FF` - A soft, light blue (Alice Blue) providing a clean backdrop for content.

 

### Typography

 

All text elements utilize the `Plus Jakarta Sans` font family, offering a modern and highly readable experience across headlines, body text, and labels.

 

*   **Headline Font:** `Plus Jakarta Sans`

*   **Body Font:** `Plus Jakarta Sans`

*   **Label Font:** `Plus Jakarta Sans`

 

### Roundedness

 

UI elements feature a maximum roundedness (`3`), giving them a soft, pill-shaped appearance. This contributes to a friendly and approachable aesthetic.

 

### Spacing

 

The design adopts a normal spacing level (`2`), providing a balanced and comfortable layout without feeling too dense or overly spacious. This ensures good readability and visual hierarchy.
